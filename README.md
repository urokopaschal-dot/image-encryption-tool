# Image Encryption Tool

This project encrypts and decrypts images using pixel manipulation in Python.

## Features
- Encrypt images using a secret key
- Decrypt encrypted images
- Simple pixel value modification technique

## Technology Used
Python
Pillow Library

## How to Run

1. Install pillow

pip install pillow

2. Run the program

python image_encryptor.py

3. Choose encrypt or decrypt and enter the image name.
